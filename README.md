# ROBO2021_Project_Scratch

## Requirements

 * Python 2.7
 * Java
 * Google Closure Library  `npm i google-closure-compiler -g`

## Create dependencies

```
cd scratch-vm

npm install

npm link

npm run watch

cd ../scratch-blocks

npm install

npm link

cd ../scratch-gui

npm install

npm link scratch-vm scratch-blocks

npm start

```
